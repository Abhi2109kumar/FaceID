## üìã Overview

FaceID is designed to help you recognize faces accurately while ensuring that your system is secure from common spoofing methods such as photos and videos. It incorporates two main techniques:

1. **Active Challenge-Response:** This method checks for user engagement through actions like blinking and changing poses.
2. **Passive Texture Analysis:** This technique uses Fast Fourier Transform (FFT) frequency monitoring to analyze skin texture and detect potential fakes.

Additionally, it features local JSON biometric storage and pose-aware signature matching, ensuring that your data remains secure and easily retrievable.
